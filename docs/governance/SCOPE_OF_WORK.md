# Scope of Work

## In scope
- Snowflake data model and SQL initialization scripts.
- Temporal analytics views and state chunking logic.
- Python backend for retrieval, generation, safety, and audit events.
- Streamlit application for operations/progress/governance views.
- dbt-style model structure (sources, staging, marts).
- Deterministic offline demo execution path.
- Technical, commercial, and leadership documentation bundle.

## Out of scope
- Direct PLC integration and live plant actuator commanding.
- Vendor-specific real-time control loop writeback automation.
- Formal external certification package (ISO/SOC) issuance.
- Production MLOps retraining pipeline automation.

## Delivery artifacts
- Source code and SQL assets
- Test suite
- Visual diagrams and explainers
- Thesis and product documentation
- Governance and IP documentation pack
